Acknowledgements

This plugin builds on the work of several open-source projects and Emby community plugins.

Playback Reporting The architecture pattern for this plugin (SQLite caching, embedded web UI, scheduled tasks) is based on playback_reporting by faush01.
ChapterApi Emby's segment and chapter APIs were referenced from ChapterApi by faush01.
EmbyCredits Credits detection features integrate with the EmbyCredits plugin by rlauuzo. Install EmbyCredits to enable "Detect Credits" buttons throughout the UI.
